# Wickly

Тёплый личный дневник без аккаунта. Всё живёт на твоём устройстве, синхронизация идёт напрямую между твоими устройствами. Ноль облака, ноль рекламы, ноль трекеров.

> A warm personal journal with no account. Everything stays on your device; sync goes directly between your own devices. Free and open source.

## Идея

- **Без аккаунта.** Не нужны почта и пароль — дневник твой с первой секунды.
- **Локально и приватно.** Записи хранятся на устройстве, под замком (PIN + биометрия), с шифрованием AES-256-GCM.
- **P2P-синхронизация.** Прямое сопряжение устройств или общая папка (Syncthing) — данные не уходят в чужое облако.
- **Красиво.** Material 3 Expressive: крупные формы, шрифты Unbounded + Onest, живое движение.
- **Бесконечные темы.** Любой цвет через колор-пикер, 4 режима темы, AMOLED, Material You.
- **7 языков** интерфейса: Русский, English, Deutsch, Français, Español, Italiano, Português.

## Что умеет

**Записи.** Редактор с подсветкой markdown прямо в поле ввода, автосохранение, фото и голос внутри текста, хэштеги превращаются в теги, дата задним числом, место и погода. Запись раскладывается на темы — читалка рисует их карточками. Обложку можно подобрать по теме из Openverse (свободные лицензии, автор указывается) или взять своё фото.

**Взгляд назад.** Лента с сериями и «в этот день», календарь настроения, медиа-сетка мозаикой, карта записей, воспоминания, поиск с фильтрами.

**Настроение и привычки.** Эмоции и действия в духе Daylio, свои варианты, трекеры и привычки, статистика с корреляциями.

**Всё остальное.** Несколько дневников с обложками, напоминания с подсказками для письма, экспорт (MD, JSON, TXT, PDF-книга), шифрованный бэкап, виджет на домашний экран, диктовка и аудио-заметки.

## Статус

Приложение собрано целиком — 26 экранов. Свежие сборки лежат в [релизах](https://github.com/THET1ME-1/Wickly/releases), обновляется приложение само.

Проверено автотестами: 81 тест, снимки экранов в трёх режимах темы, отдельные прогоны слоя данных и синхронизации (включая слияние двух настоящих баз и сходимость встречных правок).

## Технологии

Flutter (Material 3), SQLite с CRDT для синхронизации без сервера. Дизайн строится на общей дизайн-ДНК: единый seed-цвет → `ColorScheme.fromSeed`, пилюли-кнопки, карточки с радиусом 28, волнистые прогресс-индикаторы.

## Сборка

```bash
flutter pub get
flutter run
```

Проверки:

```bash
flutter test                       # тесты
flutter test test_golden           # снимки экранов
dart run tool/db_smoke.dart        # слой данных
dart run tool/sync_smoke.dart      # синхронизация
```
